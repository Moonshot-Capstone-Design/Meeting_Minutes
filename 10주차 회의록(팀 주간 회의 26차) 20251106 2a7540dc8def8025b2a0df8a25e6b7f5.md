# 10주차 회의록(팀 주간 회의 26차) 20251106

날짜: 2025년 11월 6일
유형: 팀 주간 회의
AI 요약: 회의에서는 엘리베이터의 버튼 인식률과 탑승 및 하차 동작 성공률을 측정하기 위한 시나리오와 방법을 구체화하였다. 사람의 유무에 따른 인식 정확도와 통과 기준을 설정하고, 로봇의 경로 계획 및 충돌 회피 방안을 논의하였다. 또한, 하드웨어 및 소프트웨어 구성 요소에 대한 진행 상황과 향후 작업 계획이 포함되었다.
참석자: 한주형, 장윤서, 김형진, 수민 김

## <각자 맡은 역할에 대한 임무 진행>

**[형진]** 

- 구동부 전력계통 설계.
21700 배터리로 29.4v 10A사양 배터리팩 주문제작. 엠디로봇사의 md200t모터드라이버와 mdh100 모터를 사용하여 구동부 전력계통 구성
- 인휠모터 결선.
UVW3상 전원, 홀센서 UVW3상, 엔코더 등 모터 드라이버와 모터 결선 작업 수행.
배터리 탈부착 형식으로 제작
- ROS기반 모터드라이버 응용sw 설계
데이터 시트 참고하여 송수신 데이터 프레임 기반으로 sw설계
- ROS기반 모터굴리기.
키보드 입력으로 모터를 동작시키는 데모 코드 작성
- ROS기반 TTS_Speaker 설계.
외부 트리거를 구독해서 조건에 맞는 녹음 파일을 재생하는 코드 작성

**[수민]**

- 로봇에 포함된 각 하드웨어의 데이터 시트를 참고하여 나사 규격과 부품 배치를 설계함.
- 엘리베이터 버튼 인식을 위해 학습용 데이터를 수집함.
- 로봇이 요구 하중을 견딜 수 있도록 형상 설계 및 인필(infill) 설정을 최적화하고, 이를 기반으로 모델링 및 출력 품질을 관리함.

[윤서]

- 로봇 기구 본체 전 영역의 3D 모델링(Fusion 360) 총괄
- 1층(저층: 구동/전원부), 2층(상층: 라이다/IMU센서부/프로파일고정부 등) 2단 구조 설계 및 지속 수정
- 모듈화된 브래킷/하우징 일체 설계(모터, 배터리, 메인보드, 라이다, 케이블)
- 3D 프린팅(부품 출력, 공차 보정, 조립성 검증) 실무 담당

[주형]

- 엘리베이터 버튼 인식을 위한 학습 데이터 셋 수집 및 라벨링 진행
- 라벨링을 거친 데이터 셋을 통해 yolob8 nano, small 등 로보플로우에서 지원하는 다양한 버전의 모델로 생성함. 생성된 모델들의 성능을 비교하여 가장 우수한 모델을 선별하고, 해당 데이터 셋과 모델을 통해 Colab 환경에서 .pt 학습 파일 생성
- 사람 객체 인식  .pt 학습 파일 생성
- 엘리베이터 버튼 인식 + 사람 객체 인식 학습 파일을  ROS 환경에 올린 후, 각각의 라벨링에 맞는 텍스트를 추출 → 토픽 발행 하여 시각화 할 수 있도록 적용
- 카메라 조도 및 채도 조정을 통해 수집한 데이터 셋과 최대한 비슷한 카메라 송출 화면을 구성하여 객체 인식률 향상.
- 조건부 객체 인식 환경(버튼 인식 → 사람 인식 → 버튼 미인식 → 음성출력)에 따른 TTS_Speaker 송출 설정.
